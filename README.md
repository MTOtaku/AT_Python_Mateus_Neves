Projeto Python Feito para trabalho da faculdade

Dependencias : Python, Pandas, beautifulsoup4, sqlalchemy

O sistema coleta filmes do IMDB e cria um database para se colocar os filmes listados nele, também cria arquivos JSON para salvar e ler informações dos filmes ou series, junto com o config.json que estpa o link do site

É possivel filtrar os filmes por nota e ordernar os mesmos por nota > 9.0 como pedido nos exercicios, No codigo possui-se 2 Series ficticias criadas para demonstrar que também é possivel adicionar series manualmente

O codigo salva arquvivos CSV e JSON dos filmes / series, também é adicionado no database para leitura por SQL
